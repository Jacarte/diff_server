# Github diff proxy

Returns diff url file in a REST API call

Deployed in ```http://jacarte.pythonanywhere.com```

```JS
fetch(`http://jacarte.pythonanywhere.com/?url=${diffURL}`, {method: 'GET'})
      .then(function (data) {
          data.json().then(json => {
              console.log(json);
          }).
          catch(err => {

                  console.log(err);
              }
          );
      })
      .catch(function (error) {
      
      });

```
